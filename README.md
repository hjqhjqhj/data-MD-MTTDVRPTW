DATASET README
Instances Included: C101, RC101, RC102
======================================================================

1. DATA STRUCTURE
----------------------------------------------------------------------
Each instance file contains exactly 101 rows of data:
* Row 1:        Distribution Center (Depot) information.
* Rows 2 - 101: Information for 100 Customers.

2. COLUMN DEFINITIONS
----------------------------------------------------------------------
Each row consists of 10 columns. The fields are defined as follows:

* Column 1:   Index
* Column 2:   X-Coordinate
* Column 3:   Y-Coordinate
* Column 4:   Left Time Window 
* Column 5:   Right Time Window 
* Column 6:   none
* Column 7:   Service Time
* Column 8:   Demand for Product Type 1
* Column 9:   Demand for Product Type 2
* Column 10:  Demand for Product Type 3

DATASET README
Instances Included: shili.txt (Multi-Depot Cold Chain Routing)
DATA STRUCTURE

Each instance file contains exactly 22 rows of data:

Rows 1 - 2:   Distribution Centers (Depots) information (2 Depots).

Rows 3 - 22:  Information for 20 Customers.

COLUMN DEFINITIONS

Each row consists of at least 11 columns. The fields are defined as follows:

Column 1:   Index

Column 2:   X-Coordinate (Longitude)

Column 3:   Y-Coordinate (Latitude)

Column 4:   Left Time Window (Earliest start/arrival time)

Column 5:   Right Time Window (Latest end/arrival time)

Column 6:   Service Time

Column 7:   none

Column 8:   none

Column 9:   Demand for Product Type 1

Column 10:  Demand for Product Type 2

Column 11:  Demand for Product Type 3
